StoreFlow is a modern e-commerce management system designed to help store owners organize products, variants, inventory, and product attributes from one simple dashboard. The system allows each product to have multiple
  variants, such as different colors, sizes, prices, SKUs, and quantities. It also supports linking several attribute values to the same variant, for example combining “Red” with “Size L”.

  The platform provides a clean admin interface connected directly to the backend API, making it easy to create products, manage stock, define attributes, and build flexible product catalogs efficiently.
